# 💳 Payment Card UI (HTML & CSS)

A clean and responsive **payment/order summary card UI** built using pure **HTML and CSS**. This project demonstrates modern layout techniques like **Flexbox, gradients, and hover animations**.

---

## 🚀 Overview

This project creates a **centered card layout** that mimics a real-world payment interface (similar to Razorpay/Spotify subscription UI).

---

## 🎯 Features

- 🎨 Gradient background
- 📦 Centered card layout using Flexbox
- 🖼️ Image integration (payment + logo)
- 💳 Order summary section
- 💰 Pricing display
- 🔘 Payment button with hover animation
- ❌ Cancel option with hover effect

---
